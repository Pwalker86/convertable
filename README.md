# Convertable

Convertable converts like a super awesome converter

### Installation

```ruby
  gem 'convertable'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install convertable

### Examples

`20.feet` => 6.096 meter

`20.feet.value` => 6.096

`20.feet.unit` => meters

`20.feet.base_meter` => 6.096

`20.feet.to_centimeters` => 609.6 centimeter

### Running the Gem in Console

`ruby bin/console`

### Running the Tests

`rake`
